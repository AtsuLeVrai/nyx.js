## Framework Features

Nyxo.js provides a set of features to make Discord bot development easier:

- **Command Handling** - Create and manage slash commands with ease
- **Event Handling** - Respond to Discord events in a structured way
- **Middleware Support** - Add reusable middleware to commands and events
- **Error Handling** - Robust error handling to keep your bot running smoothly
- **Type Safety** - Full TypeScript support for a better development experience
- **Plugin System** - Extend functionality with a modular plugin system
