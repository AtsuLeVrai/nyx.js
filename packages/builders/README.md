## 🚀 About

`@nyxojs/builders` provides a comprehensive set of builder classes for constructing Discord API components with ease.
This package includes type-safe, fluent API builders for creating embeds, buttons, select menus, action rows, and more.

> [!NOTE]
> This package is part of the Nyxo.js ecosystem but can be used independently in any Discord bot or Node.js application.
